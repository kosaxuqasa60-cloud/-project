## 1. 全局框架与导航

![资源平台首页](/prd/01-resource-home.png)

**需求说明**

- 采用「左侧一级导航 + 顶栏 + 主工作区」的三段式布局。
- 左侧一级导航固定为：首页、备课、教学、作业、错题、学情、**资源**、管理；当前模块「资源」高亮。
- 顶栏展示产品名「有我知了 · 教学评一体化AI空间」、学校（有我初中）、教师账号（xh200）、学科切换（数学）、通知与安全入口。
- 主工作区左栏为教材与章节导航，右栏为资源列表；整体绿色主题色，符合教育场景稳重、清晰的调性。

## 2. 资源库检索

### 2.1 教材上下文与切换

![教材切换弹窗](/prd/04-textbook-switcher.png)

**需求说明**

- 左栏顶部固定「教材卡」：显示当前教材封面、学科·版本（数学·沪教版）、年级学期（七年级下册·2025学年），并提供「切换教材 / 学期」入口。
- 点击后弹出**切换教材**弹窗：
  - 支持按**学年**（2025 / 2024 学年）与**学期**（上 / 下学期）切换；
  - 教材以封面网格展示，选中项右上角打勾并高亮描边；
  - 底部「取消 / 确定切换」，切换后资源列表按新教材上下文刷新。

**需求说明**

- 左栏提供「章节目录 / 知识点」两种组织维度，可一键切换。
- **章节目录**：按教材单元层级展开（如「第15章 一元一次不等式」下含 15.1 / 15.2 / 15.3），每个节点右侧显示题量；支持父节点展开/收起，子节点选中高亮。
- **知识点**：按知识领域组织（数与代数、图形与几何、综合与实践），下挂具体知识点及题量。
- 选中某章节/知识点后，右侧列表联动过滤对应资源。

### 2.3 来源与题型难度筛选

![更多筛选展开](/prd/03-more-filters.png)

**需求说明**

- 列表顶部提供**来源**筛选：全部 / 市级 / 区级 / 校级 / 我的，用于区分题目的来源层级与归属。
- 「更多筛选」展开后出现二级筛选：**题型**（全部/单选/多选/填空/判断/主观）与**难度**（全部/易/中/难）。
- 右上角展示当前结果总数（如「共 5 题」）并提供**列表 / 网格**视图切换。
- 顶部标签页可切换资源类型：题目 / 作业 / 备课 / 微课 / 精品。
- 搜索框支持按题干、知识点、标签检索；右侧为「新增题目」主按钮。

### 2.4 题目卡片与详情

![题目展开详情](/prd/02-question-expand.png)

**需求说明**

- 每张题目卡片展示：序号、题型徽标、来源徽标（市/区/校）、难度、是否含视频讲解，以及题干（支持数学公式渲染）与选项。
- 卡片底部展示知识点标签、教学用途等标签，右侧提供「展开详情 / 加入（题篮）」操作。
- 展开详情后分「基本信息 / 系统标注」两个子页：
  - **基本信息**：显示【答案】【解析】（公式渲染），并关联讲解视频卡片（时长、播放入口）。
- 卡片右上角显示使用数据（如「组卷 326 · 已练 1280 人」），辅助教师判断题目质量。

## 3. 新增题目

### 3.1 选择章节（第一步）

![新增题目第一步](/prd/09-newq-step1.png)

**需求说明**

- 新增题目采用两步向导，顶部提供步骤条（1 选择章节 → 2 录入题目）。
- 第一步先确定题目归属：展示当前学科上下文徽标（数学 / 七年级下 / 沪教版），选择**单元**（列表单选）与**章节**（标签单选）。
- 完成后点击「下一步 · 录入题目」进入第二步。

### 3.2 录入题目（第二步）

![新增题目第二步](/prd/10-newq-step2.png)

**需求说明**

- 左侧主表单：
  - **基础信息**：显示已选章节徽标，可「返回修改章节」；**题目类型**支持单选/多选/判断/填空/主观/连线/组合题。
  - **核心内容**：题干（必填，支持图片与公式）、答案选项（A/B/C/D）、解析（支持「AI 生成解析」）、难度（易/中/难）。
  - **讲解资源**：可上传讲解视频、解析音频，或生成讲解脚本，沉淀为精品讲解资源。
- 右侧为标签体系（见 3.3）。底部「保存题目」。

### 3.3 标签体系与 AI 生成标签

![AI 生成标签](/prd/11-newq-ai-tags.png)

**需求说明**

- 右栏标签体系采用**受控词表**，按命名空间分组：知识点（必填）、核心素养、认知层级、教学用途、情景属性、常见错因。
- 每个分组可折叠，标签以点选方式增删，已选数量实时统计。
- 支持「AI 生成标签」：点击后系统根据题干自动推荐并勾选标签（带 AI 标识），教师可在推荐基础上二次调整，支持「重新生成」。
- 底部支持**自定义标签**：输入后回车创建，补充受控词表之外的个性化标签。

## 4. 题篮与组卷

### 4.1 题篮

![题篮抽屉](/prd/08-cart-drawer.png)

**需求说明**

- 全局右下角悬浮「题篮」入口，带数量角标；点击从右侧滑出抽屉。
- 题篮内按**题型**分组展示已选题目（一、单项选择题 / 三、填空题 …），每题显示题干摘要与来源/难度徽标，支持单题删除。
- 底部提供「清空题篮」与「生成练习」主按钮，进入组卷编辑台。

### 4.2 组卷编辑台

![组卷编辑台](/prd/12-composer.png)

**需求说明**

- 顶部展示练习标题、题量、总分、布置对象数量，右侧「存为草稿 / 布置作业」。
- 中间为试卷主体：按大题分组，题卡显示题型/来源/难度/分值、题干与选项，支持「展开详情 / 添加题目音频 / 添加跟进练习 / 移出」。
- 右侧为工具轨（作业信息 / 分值设置 / 题型及排序）三个抽屉切换入口。

## 6. 点阵笔排版框选

### 6.1 框选主界面

![框选主界面](/prd/19-dotpen-layout.png)

**需求说明**

- 页面为「确认题目与作答区」，用于将点阵笔纸质卷面与题目结构进行绑定。
- 顶栏：作业标题与切换、纸张尺寸（A4）、识别状态（如「已识别 5 题，2 处待确认」），以及保存 / 预览打印 / 下载PDF / 发布作业。
- 中间为卷面画布：支持翻页（缩略图）、缩放、适宽、实际大小、旋转；卷面上以彩色框标出**题目区**（印刷题干）与**作答区**（手写作答）。
- 左下角图例：题目区（蓝）/ 作答区（绿）/ 未框选·待确认（红）。
- 右侧为「按题目查看 / 按层级查看」双视图。

### 6.2 智能框选与手动框选（增 / 删）

**需求说明（对应上图左栏「智能框选 / 手动框选」）**

- **AI 一键框选**：自动识别并生成题目区/作答区框。
- 「显示全部框」开关：切换卷面框的显隐。
- **手动框选**：提供「题目框 / 小题框 / 作答框」三种绘制工具；选中工具后在卷面按住拖拽即可**新增**框。
- **删除**：每个框（含 AI 自动生成与手动绘制）在悬停时于右上角显示红色 `×`，点击即**删除**该框。框选支持完整的增删闭环。

- 点击某题打开**题目信息配置**弹窗：可设置该题分值、题型、答案（如单选唯一正确项），保存后回写卷面与列表。

### 6.4 按层级查看（合并 / 拆分）

![按层级查看](/prd/21-level-view.png)

**需求说明**

- 「按层级查看」以**题目层级树**呈现结构：顶部统计「总分 / 大题数 / 小题数」。
- 层级为：大题 → 小题 → 作答区。
  - 小题显示题号徽标、题型徽标；作答区显示答案徽标（客观题）与分值。
- 支持**合并 / 拆分**（对应参考图）：
  - 小题左侧圆形「合」：与下一小题合并为多作答区结构；
  - 合并后的小题显示「拆」：可拆回独立小题；
  - 大题左侧圆形「合」：与下一大题合并；
  - 不可操作时显示灰色「−」。
- 大题支持展开/收起。

### 6.5 批量设置

![批量设置弹窗](/prd/22-batch-modal.png)

**需求说明**

- 「按题目查看」与「按层级查看」右上角的「批量设置」为**同一功能**（非复选批量），打开统一的层级表格弹窗。
- 弹窗以表格呈现全部小题，按大题分组，列为「题号 / 题型·答案 / 分值」：
  - 支持在大题级设置题型与「每作答区分值」并一键下发；
  - 支持逐小题设置题型、逐作答区录入答案与分值；
  - 大题分值、试卷总分实时汇总。
- 底部「取消 / 确定」，确定后统一回写各题配置。
